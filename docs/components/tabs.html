Tabs Component NEW

Tabs organize content into multiple sections with smooth animations. Navigate between different views with an animated cursor that slides between selections.

Features

  • 4 visual variants (solid, underlined, bordered, light)
  • 6 color schemes matching your theme
  • 3 sizes (sm, md, lg)
  • Animated cursor with spring physics (60fps+)
  • Icon support for tabs
  • Full width tabs option
  • Disabled tabs support
  • Controlled and uncontrolled modes
  • Top/bottom placement
  • Powered by React Native Reanimated

Basic Usage

import { Tabs, Tab } from 'react-native-heroui';

<Tabs variant="solid" color="primary">
  <Tab tabKey="profile" title="Profile">
    <Text>Profile content</Text>
  </Tab>
  <Tab tabKey="settings" title="Settings">
    <Text>Settings content</Text>
  </Tab>
  <Tab tabKey="notifications" title="Notifications">
    <Text>Notifications content</Text>
  </Tab>
</Tabs>

Variants

Solid (Default)

Filled background with animated cursor sliding behind tabs.

<Tabs variant="solid" color="primary">
  <Tab tabKey="1" title="Tab 1">Content 1</Tab>
  <Tab tabKey="2" title="Tab 2">Content 2</Tab>
</Tabs>

Underlined

Clean minimal design with sliding underline.

<Tabs variant="underlined" color="secondary">
  <Tab tabKey="1" title="Tab 1">Content 1</Tab>
  <Tab tabKey="2" title="Tab 2">Content 2</Tab>
</Tabs>

Bordered

Outlined style with border color change on selection.

<Tabs variant="bordered" color="success">
  <Tab tabKey="1" title="Tab 1">Content 1</Tab>
  <Tab tabKey="2" title="Tab 2">Content 2</Tab>
</Tabs>

Light

Subtle background tint on selection.

<Tabs variant="light" color="warning">
  <Tab tabKey="1" title="Tab 1">Content 1</Tab>
  <Tab tabKey="2" title="Tab 2">Content 2</Tab>
</Tabs>

Controlled Tabs

function MyTabs() {
  const [selected, setSelected] = useState('profile');

  return (
    <Tabs
      selectedKey={selected}
      onSelectionChange={setSelected}
    >
      <Tab tabKey="profile" title="Profile">...</Tab>
      <Tab tabKey="settings" title="Settings">...</Tab>
    </Tabs>
  );
}

Animation Details

The Tabs component features smooth animations powered by React Native Reanimated:

  • Animated Cursor: Smoothly slides between tabs with spring physics
  • 60fps+ Performance: Runs on UI thread
  • Spring Configuration: damping: 20, stiffness: 200
  • Natural Feel: Matches native platform animations

Props Reference

Prop Type Default Description
