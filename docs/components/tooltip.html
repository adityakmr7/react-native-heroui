Tooltip Component NEW

Display brief, informative messages that appear when users interact with elements. Optimized for mobile with press and long-press interactions.

Features

  • 12 placement options (top, bottom, left, right + variants)
  • 6 color schemes with theme integration
  • 3 sizes (sm, md, lg)
  • Arrow support pointing to trigger element
  • Smooth Reanimated animations (60fps+)
  • Press or long-press activation
  • Controlled and uncontrolled modes
  • Custom content support
  • Overlay background option
  • Full TypeScript support

Basic Usage

import { Tooltip, Button } from 'react-native-heroui';

<Tooltip content="Click to save your work" placement="top" showArrow>
  <Button colorScheme="primary" onPress={() => save()}>
    Save
  </Button>
</Tooltip>

Action Types

Long Press (Default)

<Tooltip content="Tooltip hint">
  <Button onPress={handleClick}>Click me</Button>
</Tooltip>

// Quick tap → Button action fires
// Long press → Tooltip appears

Press

<Tooltip content="Info text" actionType="press">
  <Button>Info</Button>
</Tooltip>

// Tap → Tooltip appears

Variants

Colors

<Tooltip content="Success!" color="success" showArrow>
  <Button colorScheme="success">Save</Button>
</Tooltip>

<Tooltip content="Warning!" color="warning" showArrow>
  <Button colorScheme="warning">Caution</Button>
</Tooltip>

<Tooltip content="Danger!" color="danger" showArrow>
  <Button colorScheme="danger">Delete</Button>
</Tooltip>

Placements

<Tooltip content="Top" placement="top">...</Tooltip>
<Tooltip content="Bottom" placement="bottom">...</Tooltip>
<Tooltip content="Left" placement="left">...</Tooltip>
<Tooltip content="Right" placement="right">...</Tooltip>

// Also supports: top-start, top-end, bottom-start, bottom-end,
// left-start, left-end, right-start, right-end

Props

Prop Type Default Description
content ReactNode required Tooltip content
children ReactElement required Trigger element
color TooltipColor "default" Color theme
size TooltipSize "md" Size variant
placement TooltipPlacement "top" Position
showArrow boolean false Show arrow
actionType "press" | "longPress" "longPress" Trigger method
withOverlay boolean true Show backdrop
