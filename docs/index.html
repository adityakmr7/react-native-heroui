Beautiful, modern UI component library for React Native

React Native TypeScript 22+ Components Dark Mode Accessible
Components Installation 🌳 Tree-Shaking Features GitHub NPM

⚡ Quick Start

npm install react-native-heroui
# or
yarn add react-native-heroui

# Then import and use
import { HeroUIProvider, Button } from 'react-native-heroui';

<HeroUIProvider>
  <Button colorScheme="primary">Get Started</Button>
</HeroUIProvider>

Get started with React Native HeroUI in seconds

npm install react-native-heroui
# or
yarn add react-native-heroui
# or

Basic Usage

import { HeroUIProvider, Button } from 'react-native-heroui';

function App() {
  return (
    <HeroUIProvider>
      <Button colorScheme="primary">Click Me</Button>
    </HeroUIProvider>
  );
}
