Custom Colors

Complete control over your app's visual identity with custom color theming

Introduction

React Native HeroUI now supports complete custom color theming. You can override any color in the theme system, use preset themes, or create your own brand colors. 

import { HeroUIProvider } from 'react-native-heroui';

const customColors = {
  light: {
    primary: '#FF6B35', // Your brand primary
    secondary: '#004E89', // Your brand secondary
    success: '#28A745',
    warning: '#FFC107',
    danger: '#DC3545',
  },
  dark: {
    primary: '#FF8A65', // Dark mode variant
    secondary: '#1976D2',
    success: '#4CAF50',
    warning: '#FFB74D',
    danger: '#F44336',
  },
};

<HeroUIProvider customColors={customColors}>
  <App />
</HeroUIProvider>

// Partial override - only change specific colors

  light: {
    primary: '#4F46E5',        // Only change primary
    'primary-500': '#4F46E5',  // And its 500 variant

Preset Themes

Use professional pre-built themes for quick setup 

import { PRESET_THEMES } from 'react-native-heroui';

// Available presets: modernBlue, purple, green
<HeroUIProvider customColors={PRESET_THEMES.modernBlue}>
  <App />
</HeroUIProvider>

// Dynamic theme switching
const [currentTheme, setCurrentTheme] = useState('modernBlue');

const themes = {
  modernBlue: PRESET_THEMES.modernBlue,
  purple: PRESET_THEMES.purple,
  green: PRESET_THEMES.green,
};

<HeroUIProvider customColors={themes[currentTheme]}>
  <App />
</HeroUIProvider>

Brand Color Helpers

Generate complete color scales from your brand colors 

import { createBrandColors } from 'react-native-heroui';

// Create brand colors with helper function
const brandColors = createBrandColors(
  '#FF6B6B', // Primary brand color
  '#4ECDC4', // Secondary brand color
  '#45B7D1', // Success color
  '#FFA726', // Warning color
  '#EF5350'  // Danger color
);

const customColors = {
  light: brandColors,
  dark: {
    ...brandColors,
    // Adjust for dark mode
    primary: '#FF8E8E',
    secondary: '#6ED5CD',
  },
};

Available Color Properties

You can customize any of these color properties

Layout Colors

background, foreground, divider, accent, focus

Content Colors

content1, content2, content3, content4

Base Colors

default, primary, secondary, success, warning, danger

Color Scales

primary-50 to primary-900, secondary-50 to secondary-900, etc.

Best Practices

Tips for creating beautiful and accessible custom themes

Always provide both light and dark variants

For better user experience and accessibility

Test your colors in both modes

Ensure sufficient contrast and readability

Use semantic color names

primary, success, etc. rather than specific colors

Start with presets

Use preset themes as a starting point for customization

Consider accessibility

Ensure sufficient contrast ratios for all users

Use brand color helpers

For consistent color scales and professional results
